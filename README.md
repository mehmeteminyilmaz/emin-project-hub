# 🚀 Emin's Project Hub | Personal Engineering Dashboard

A high-end, minimalist dashboard built to showcase a curated list of software and hardware projects. Featuring real-time status tracking and visual data analytics.

## ✨ Key Features
- **Project Tracking:** Dynamic status indicators for ongoing and completed projects (e.g., AgroLens, Kalorilens).
- **Tech Stack Analytics:** Interactive PieCharts using **Recharts** to visualize language and platform distribution.
- **Modern UI/UX:** Built with **Next.js 15**, **Tailwind CSS**, and **Glassmorphism** design principles.
- **Responsive Design:** Optimized for seamless viewing across mobile, tablet, and desktop devices.

## 🛠️ Tech Stack
- **Frontend:** Next.js (App Router), TypeScript
- **Styling:** Tailwind CSS (Glassmorphism UI)
- **Visualization:** Recharts
- **Icons:** Lucide React

## 🚦 Getting Started
1. Clone the repo: `git clone https://github.com/mehmeteminyilmaz/my-dashboard.git`
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
